## Описание игры и управления
Игра настольный теннис для двух игроков на одном устройстве

Левый игрок:
'W' - движение ракетки вверх
'S' - движение ракетки вниз

Правый игрок:
'↑' (cтрелка вверх) - движение ракетки вверх
'↓' (cтрелка вниз) - движение ракетки вниз

'ESC' - выход из игры


## Правила игры
1. Побеждает тот игрок, который первым набрал 5 очков
2. После каждого удара мяч ускоряется
3. Отклонение мяча зависит от места удара по ракетке


## Инструкция по сборке
Windows 11 Домашняя
Visual Studio Code
Запускать через 'Developer Command Prompt for VS 2022'

cd /d C:\Users\user\Desktop\TennisPong
cl /nologo /W3 /O2 /Fe:PongGame.exe main.c win32_ui.c game_logic.c user32.lib gdi32.lib msimg32.lib 
PongGame.exe



## Реализованные модули
game_logic - физика, коллизии, ускорение мяча, счёт, победа
win32_ui - окно, двойная буферизация, отрисовка GDI, ввод с клавиатуры
